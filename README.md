## 版本

提交内容中有两个版本：

App_Pre：仅支持普通加减乘除取余操作

App：支持次方三角函数等科学运算

## 说明

打开.exe文件需要电脑装有Java环境

不支持在单次运算sin、cos、tan、根号同时出现两个或其中一个出现多次
